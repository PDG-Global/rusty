## Overview

Slash commands are special commands typed in the input box, starting with `/`. They are available in both TUI mode and stdin REPL mode.

## Available Commands

| Command | Alias | Description |
|---------|-------|-------------|
| `/help` | `/h`, `/?` | List all available commands |
| `/init` | | Generate an AGENTS.md file for the current codebase |
| `/resume` | `/r` | Resume a saved session (interactive picker in TUI) |
| `/sessions` | `/s` | List all saved sessions |
| `/compact` | | Force conversation compaction |
| `/clear` | | Clear the screen |
| `/copy` | `/c` | Copy the last assistant response to clipboard |
| `/model` | `/m` | Show the current model |
| `/rename` | | Rename the current session |
| `/settings` | | Open the settings overlay |
| `/permissions` | | Manage tool permissions |
| `/version` | `/ver`, `/v` | Show the running version and check for updates |
| `/quit` | `/exit`, `/q` | Exit Rusty (saves the session) |

## Command Details

### /help

Displays a list of all available commands with brief descriptions. Aliases: `/h`, `/?`.

### /init

Generates an `AGENTS.md` file tailored to the current codebase. This file provides context about the project structure, conventions, and key files to help the AI understand the project better.

### /resume

Opens an interactive session picker (in TUI mode) or lists sessions for selection (in stdin mode). Use this to continue a previous conversation. Alias: `/r`.

### /sessions

Lists all saved sessions with their IDs, models, and timestamps. Alias: `/s`.

### /compact

Forces conversation compaction. This summarizes older messages to reduce context window usage. Normally, compaction triggers automatically when the conversation exceeds approximately 80,000 tokens or 40 messages. Use this command to trigger it manually.

### /clear

Clears the terminal screen. The conversation history is preserved; only the display is cleared.

### /copy

Copies the last assistant response to the system clipboard. Alias: `/c`.

### /model

Displays the currently active model name and provider. Alias: `/m`.

### /rename

Renames the current session. Prompts for a new name in TUI mode.

### /settings

Opens the settings overlay in TUI mode, where you can review and adjust configuration interactively. In headless stdin mode, settings are not available — edit `~/.rusty/settings.json` directly.

### /permissions

Opens the tool permissions manager. Use this to review which tools are currently allowed or denied and to change permission rules for the active session.

### /version

Prints the running Rusty version and checks GitHub for a newer release. If an update is available, the output includes the latest version number and a link to the releases page. Aliases: `/ver`, `/v`.

```
> /version
rusty v0.1.6
Update available: v0.1.7
https://github.com/PDG-Global/rusty/releases
```

Rusty also runs this check automatically in the background when the TUI starts. If a newer version is found, a yellow update notification appears in the status bar. The startup check is non-blocking and times out after 5 seconds, so a slow or unreachable network will not delay startup.

### /quit

Exits Rusty. The current session is saved automatically before exiting. Aliases: `/exit`, `/q`.

## Autocomplete

In TUI mode, slash commands are completed through an interactive dropdown. As soon as you type `/`, a popup appears above the input box listing every command whose name starts with what you have typed so far, along with a short description. The list filters live as you keep typing.

### Key bindings

| Key | Action |
|-----|--------|
| `↑` / `↓` | Move the selection up or down within the dropdown |
| `Tab` | Accept the highlighted command and insert it into the input, without submitting |
| `Enter` | Accept the highlighted command and submit it immediately |
| `Esc` | Dismiss the dropdown without changing the input |

Use `Tab` when you want to inspect or extend the command before running it (for example, accept `/rename` and then type the new name). Use `Enter` when the highlighted command is exactly what you want to run.

### Examples

- Type `/` to see all available commands.
- Type `/re` to narrow the dropdown to `/rename`, `/resume`. Use `↓` to highlight `/resume`, then press `Enter` to run it.
- Type `/he`, then press `Tab` to fill in `/help` without submitting, in case you want to edit it first.
- Press `Esc` at any time to close the dropdown and keep typing a regular message.
