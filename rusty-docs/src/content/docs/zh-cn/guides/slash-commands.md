title: 斜杠命令
description: 会话期间控制 Rusty 的内置命令
---


## 概述

斜杠命令是可以在输入框中输入的特殊命令（TUI 与标准输入 REPL 模式均可用），用于控制 Rusty 而不向 LLM 发送消息。输入 `/` 即可通过 tab 补全查看可用命令。

## 命令参考

| 命令 | 别名 | 说明 |
|---------|---------|-------------|
| `/help` | `/h`、`/?` | 列出所有可用命令 |
| `/init` | | 为当前代码库生成 AGENTS.md 文件 |
| `/resume` | `/r` | 恢复已保存的会话（TUI 模式下为交互式选择器） |
| `/sessions` | `/s` | 列出所有已保存的会话 |
| `/compact` | | 强制进行对话压缩 |
| `/clear` | | 清屏 |
| `/copy` | `/c` | 将最后一条助手响应复制到剪贴板 |
| `/model` | `/m` | 显示当前模型 |
| `/rename` | | 重命名当前会话 |
| `/permissions` | `/perms` | 查看或撤销「始终允许」的工具权限 |
| `/settings` | | 打开模型注册表与通用设置浮层（仅 TUI） |
| `/cost` | | 显示 token 用量与预估费用 |
| `/quit` | `/exit`、`/q` | 退出（会保存会话） |

## 命令详解

### /compact

强制进行对话压缩。通常压缩会在上下文窗口占用达到 25%、50%、75% 时自动触发。当你希望在触及自动阈值之前腾出上下文空间时，可用此命令手动压缩。

### /copy

将最后一条助手响应复制到系统剪贴板。方便把代码建议粘贴到编辑器中。

### /model

显示当前活动的模型名称、提供方与 API base URL。

### /permissions

显示当前权限状态：

- **会话允许列表**：仅本次会话内被批准的工具
- **永久允许列表**：被永久批准的工具（持久化在 `~/.rusty/settings.json`）

你也可以在此视图中撤销特定的「始终允许」权限。

永久允许列表条目使用 `tool_name:exact_input` 格式。例如：

```
bash:git status
bash:cargo check
file_read:src/main.rs
```

### /settings

打开设置浮层（仅 TUI 模式）。这提供了一个可视化编辑器，用于：

- **模型注册表**：浏览并切换已配置的模型、编辑模型条目
- **通用设置**：权限模式、思考等级及其他偏好

### /cost

显示当前会话的 token 用量统计：

- 消耗的输入 token
- 生成的输出 token
- 思考 token 总数
- 预估费用（基于设置中配置的每模型定价）

### /resume

在 TUI 模式下，显示交互式选择器以浏览并选择已保存的会话。在标准输入 REPL 模式下，接受会话 ID 作为参数。

### /init

生成一份针对当前代码库定制的 AGENTS.md 文件。该文件帮助 LLM 理解项目结构、编码约定与常见模式。生成的文件放在项目根目录。

### /rename

重命名当前会话。接受一个新名称作为参数。会话名称让你之后使用 `/resume` 或 `/sessions` 时更容易找到会话。
