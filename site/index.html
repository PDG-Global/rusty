What it does

An agent that reads your code, runs commands, edits files, and reasons through problems. Works with any OpenAI-compatible API endpoint.

terminal

Shell execution

Runs commands with a tiered permission system. Read-only operations auto-approve. Write operations prompt you inline.

files

File operations

Read, write, and edit files with surgical precision. Shows you what changed before committing.

reason

Transparent reasoning

Visible thinking tokens let you watch the agent work through problems step by step. No black boxes.

secure

Sandboxed by default

File access is restricted to your working directory. Destructive commands and write operations require explicit approval.

Get started in seconds

One command to install. Point it at your model. That's it.

zsh
# Install rusty
$ curl -sSf https://rusty.dev/install.sh | sh
 
# Point it at your model
$ export OPENAI_API_BASE=https://api.deepseek.com/v1
$ export OPENAI_API_KEY=your-key
 
# Start coding
$ rusty

Built on principles, not features.

  • 01 One binary, zero config. Drop it in your PATH and go. No package managers, no runtime installs, no config files to edit.
  • 02 Model-agnostic by design. Works with any OpenAI-compatible endpoint. Switch models by changing an environment variable.
  • 03 Your keys, your data. No telemetry. No proxy. No account required. Your API key stays local.
  • 04 Security is the default. File access is sandboxed. Destructive operations require explicit approval. You see every command before it runs.
